About Wembedder

Wembedder is a Wikidata knowledge graph embedding webapplication and webservice. On the most-similar page a user can search in multiple languages for a Wikidata item and Wembedder will return a ranked list of the most similar other Wikidata items.

An API is available. For instance, /api/most-similar/Q18618629 will return JSON for a most-similar query on the item Q18618629 (Denny Vrandečić), while /api/vector/Q18618629 contains the embedding vector for Q18618629 and /api/similarity/Q34/Q35 will return JSON with the numerical similarity between Q34 and Q35 Wikidata items.

Development

Wembedder is developed under the Apache License 2.0 license at GitHub. The code can be downloaded and run locally. Pre-trained models are necessary.

Pre-trained models trained with a simplified RDF2vec approach are available at Zenodo:

  • Wembedder wikidata-20170613-truthy-BETA-cbow-size=100-window=1-min_count=20
    Wikidata embedding Gensim CBOW trained on truthy dump with a 100-dimensional subspace, a window of 1 and a minimum count of 20. https://zenodo.org/record/823195
  • Wembedder wikidata-20170613-truthy-BETA-cbow-size=100-window=1-min_count=20-iter=25
    Wikidata embedding Gensim CBOW trained on truthy dump with a 100-dimensional subspace, a window of 1 and a minimum count of 20. Trained with 25 iterations. https://zenodo.org/record/827339
